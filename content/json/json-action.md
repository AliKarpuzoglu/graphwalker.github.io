A single GraphWalker action representation in JSON

## action

```json
  "<JAVA SCRIPT;>"
```

The action is a piece of java script code that you want the model to execute.

It has to end with a semi colon.

## Example

This is an example of an action.

```json
  "numOfPets=0;"
```
